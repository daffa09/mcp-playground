# Proyek API E-Commerce

REST API yang kokoh untuk aplikasi e-commerce yang dibangun dengan Node.js, Express, dan TypeScript.

## Fitur

- ✅ Autentikasi & otorisasi pengguna.
- ✅ Manajemen produk (operasi CRUD).
- ✅ Keamanan berbasis JWT.
- ✅ Validasi input.
- ✅ Enkripsi kata sandi.
- ✅ Lapisan abstraksi database.
- ✅ Dokumentasi API yang komprehensif.

## Tech Stack

- **Runtime:** Node.js
- **Bahasa:** TypeScript
- **Framework:** Express.js
- **Database:** PostgreSQL
- **Autentikasi:** JWT
- **Validasi:** Custom middleware

## Memulai Cepat (Quick Start)

Lihat [SETUP.md](./docs/SETUP.md) untuk instruksi instalasi yang mendalam.

```bash
npm install
npm run dev
```

## Struktur Proyek

- `/api` - Handler rute API.
- `/services` - Lapisan logika bisnis.
- `/middleware` - Middleware Express (autentikasi, validasi).
- `/utils` - Fungsi utilitas (kripto, validasi).
- `/config` - File konfigurasi.
- `/docs` - Dokumentasi.
